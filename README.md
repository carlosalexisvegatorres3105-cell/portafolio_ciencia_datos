Portafolio de Ciencia de Datos & Analítica Avanzada

Bienvenido a mi espacio profesional. Este repositorio consolida las fuentes de datos, los desarrollos analíticos, las visualizaciones interactivas y la documentación estratégica creados durante mi formación en Ciencia de Datos con EBAC.

📂 Estructura del Repositorio

📊 1. Directorio /datos (Fuentes de Información)

Contiene los conjuntos de datos que sirvieron de base para los proyectos entregables:

    DIM_CALENDAR.xlsx
    
    DIM_CATEGORY.csv

    DIM_PRODUCT.xlsx

    DIM_SEGMENT.xlsx

    FACT_SALES.csv  

🛠️ 2. Directorio /codigo (Desarrollo Técnico)

Soluciones analíticas, consultas estructuradas y cuadernos de modelado:

    Proyecto empresa aliada entregable 1.ipynb y 2.ipynb: Fases de ingeniería de características, limpieza de datos y modelado inicial.  

    Avance de Proyecto parte 1 y 2.ipynb: Aplicación de técnicas de aprendizaje automático para el reconocimiento de dígitos escritos a mano (dataset MNIST e imagen personalizada en Paint), comparando los modelos de clasificación k-NN (k=14, precisión: 0.9567) y Regresión Logística (precisión: 0.8978).  

    Avance de Proyecto parte 3.ipynb: Segmentación analítica de clientes mediante métricas RFM (Recency, Frequency, Monetary) y el algoritmo de aprendizaje no supervisado k-Means, clasificando la base de datos en los segmentos Low-Value, Average, Potential y High-Value.  

    Avance de Proyecto parte 4.sql: Script formal con 15 consultas estructuradas sobre la base de datos dvdrental para resolver preguntas de gestión, aplicando filtros, uniones de tablas (JOIN) y funciones de agregación.  

    proyecto_final.ipynb: Pipeline maestro definitivo de ciencia de datos que integra la carga, limpieza y unión de tablas en un DataFrame unificado de más de 152,000 registros, aplicando un clustering K-Means de 3 segmentos y modelos predictivos de series de tiempo (ARIMA para la marca Vanish y Regresión Lineal para Lysol).  

    Dashboard Power BI Proyecto 5.pdf: Reporte visual interactivo de inteligencia de negocios para la exploración dinámica de ventas.  

🖼️ 3. Directorio /resultados (Evidencia Visual y Gráficos)

Recopilación de las visualizaciones y dashboards clave generados para la interpretación de los hallazgos de negocio.

📄 4. Directorio /documentos (Reportes Ejecutivos)

Compilación de la documentación formal en formato PDF alineada con el método científico:  

    Avance de Proyecto parte 1 y 2.pdf: Reporte técnico sobre el reconocimiento de dígitos escritos a mano y análisis comparativo de modelos supervisados.  

    Avance de Proyecto parte 3.pdf: Sustento metodológico de la segmentación comercial basada en RFM y clustering K-Means.  

    Avance de Proyecto parte 4.pdf: Documentación analítica y resultados de las consultas DQL ejecutadas en la base dvdrental.  

    Proyecto Final.pdf: Reporte de cierre que consolida el modelo maestro, análisis regional, clustering estratégico y recomendaciones predictivas.  

    Proyecto empresa aliada entregable 5.pdf: Reporte ejecutivo final con el plan de consultoría aplicado a la problemática del negocio real.
