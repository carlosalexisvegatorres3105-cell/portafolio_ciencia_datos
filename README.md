# Portafolio de Ciencia de Datos

Este repositorio contiene la integración de mi proyecto final de certificación.

## Estructura del Repositorio
* **/codigo**: Contiene el notebook de Jupyter con el desarrollo del análisis.
* **/datos**: Almacena las bases de datos en formato Excel y CSV utilizadas.
* **/documentacion**: Reportes formales y presentación final en Word y PDF.
* **/resultados**: Archivos visuales y gráficas estadísticas generadas.
